01 Hugo

Hugo

Hugo is a static site generator written in Go and claims to be fastest framework in the world.

Features of Hugo

  • It’s fast
  • Markdown (and other formats) to static HTML
  • Themes
  • Multilanguage support
  • Builtin server for local development
  • Easy to deploy
  • Great documentation
  • Shortcodes
